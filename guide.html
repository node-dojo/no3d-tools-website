Getting Started

Install the No3d Tools addon and sync your asset library into Blender.

What You Get

Your No3d Tools subscription gives you a synced asset library inside Blender's Asset Browser. Geometry node tools, materials, mesh objects — all drag-and-drop ready. New assets are synced automatically when you open Blender.

Requirements

  • Blender 4.5 or newer
  • An active subscription — subscribe here
  • Your license key (emailed after subscribing)

Install the Addon

Step 1: Download

Download No3D Tools Addon.zip or use the link in your license key email.

Step 2: Install in Blender

1. Open Blender

2. Go to Edit → Preferences → Add-ons

3. Click Install... (top right)

4. Select the downloaded .zip file

5. Click Install Add-on

Step 3: Enable

1. Search for "No3d Tools Membership" in the add-ons list

2. Check the box to enable it

3. The preferences panel will expand

Setup Sequence

The addon preferences panel walks you through setup with numbered steps. Open it at Edit → Preferences → Add-ons → No3d Tools Membership.

Step 1. Enter your NO3D license key

Paste your key into the Key field. Format: NO3D-XXXX-XXXX-XXXX-XXXX

Step 2. Validate and set startup sync

Click Validate License. Check Sync on startup to automatically sync when Blender opens.

Step 3. Choose install location

The default location is ~/Documents/No3d Tools. Change it if you want assets stored elsewhere.

Step 4. Apply location

Click Apply Path to confirm the install folder. Assets are always stored in a "No3d Tools" subfolder.

Step 5. Run initial sync

Click Sync Now to download the entire library. First sync pulls all assets; future syncs only download what's changed.

Step 6. Refresh if library looks empty

Click Refresh Library to reload the Asset Browser if assets don't appear immediately.

Don't have your key? Look it up here or check your email for "No3d Tools — Your License Key".

Using Your Assets

Open the Asset Browser

1. In any Blender workspace, open the Asset Browser

2. In the dropdown (top left), select NO3D Tools

3. All your synced assets appear

Drag and Drop

Asset Type How to Use
Geometry Node tools Drag onto a mesh object → adds as modifier
Materials Drag onto any object → applies material
Mesh objects Drag into viewport → adds to scene
Collections Drag into viewport → instances the collection

Troubleshooting

"Invalid license key"

  • Include the NO3D- prefix
  • Check for extra spaces
  • Verify your subscription at account page

"Sync failed"

  • Check your internet connection
  • Click Validate License first, then Sync Now
  • If it persists, try Recover Setup under Maintenance Tools

Assets not showing in Asset Browser

  • Select NO3D Tools in the Asset Browser dropdown (top left)
  • Click Refresh Library (Step 6) in addon preferences
  • If still empty, run Sync Now then Refresh Library

Lost your license key

  • Check email for "No3d Tools — Your License Key"
  • Visit account page and enter your email
