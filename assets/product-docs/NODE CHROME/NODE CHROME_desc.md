# NODE CHROME

## Description

NODE CHROME is a powerful Blender geometry node asset that provides advanced chrome material and shader capabilities. This asset enables you to create realistic chrome surfaces with customizable properties, perfect for product visualization, automotive rendering, and industrial design projects.

The node system offers flexible control over chrome appearance, including reflection intensity, surface roughness, color tinting, and environmental mapping. Whether you're creating polished metal surfaces, reflective objects, or stylized chrome effects, NODE CHROME provides the tools you need.

## Features

- **Asset Type**: GeometryNodeTree
- **Blender Version**: 5.0.0+
- **Material System**: Advanced chrome shader with customizable properties
- **Reflection Control**: Adjustable reflection intensity and quality
- **Surface Properties**: Control over roughness, metallic properties, and color tinting
- **Environment Mapping**: Built-in support for environmental reflections
- **Performance Optimized**: Efficient node setup for real-time and render workflows

## Usage Instructions

### Basic Usage

1. Download the `NODE CHROME.blend` file
2. Open in Blender 5.0.0+
3. Import or drag the asset into your scene
4. Apply the material to your geometry
5. Adjust parameters in the Geometry Nodes modifier panel

### Advanced Usage

- Customize chrome properties for different surface types
- Combine with other material nodes for complex effects
- Use in procedural workflows with other geometry nodes
- Integrate with lighting setups for optimal reflections
