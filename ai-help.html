AI Help

Get expert-level AI assistance with your No3d Tools assets.

AI assistants like ChatGPT, Claude, and Gemini can help you with No3d Tools — but they don't know about our products by default. Give them context and they'll answer specific questions about parameters, workflows, and troubleshooting.

Quick Start

Paste this prompt at the start of any AI conversation:

Fetch https://no3dtools.com/llms-full.txt and use it as your reference for all No3d Tools questions in this conversation.

That's it. The AI will fetch our complete product reference and use it to answer your questions. Works with ChatGPT, Claude, Gemini, and any AI that can browse the web.

What You Can Ask

"How do I make a hex bolt with Dojo Bolt Gen?"

"Which No3d tool generates procedural brackets?"

"What does the 'a untapped' parameter do?"

"My addon sync failed — how do I fix it?"

"How do I instance Bolt Gen across multiple points in geometry nodes?"

"I need to add knurling to a cylindrical surface — what tool should I use?"

Setup by Tool

ChatGPT / Claude / Gemini (Browser)

Effort: 5 seconds · Lasts: one conversation

Paste the quick start prompt above into your conversation. The AI fetches the reference automatically.

Claude Projects (Persistent)

Effort: 1 minute · Lasts: every conversation in the project

1. Go to claude.ai → Projects → New Project

2. Name it something like "Blender / No3d Tools"

3. In the project instructions, paste:

You are helping me with Blender projects that use No3d Tools assets. Fetch https://no3dtools.com/llms-full.txt for complete product reference. Use it to answer questions about specific tools, parameters, and workflows.

Every conversation in this project will have No3d Tools expertise built in.

ChatGPT Custom GPT

Effort: 2 minutes · Lasts: permanent

1. Go to chatgpt.com → Explore GPTs → Create

2. In the instructions, paste the quick start prompt above

3. Or download the full reference file and upload it as knowledge

What the AI Gets

The reference file includes:

  • Every No3d Tools product with full descriptions
  • Parameter documentation and node group diagrams
  • Usage guides and workflow tips
  • Membership addon setup and troubleshooting
  • Asset types: how to drag-and-drop geometry nodes, materials, mesh objects

The file is updated whenever we add or update products. You don't need to re-download anything — the AI fetches the latest version each time.
